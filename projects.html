2. Loot Manager (Group Project)

With its comprehensive features, Loot Manager aims to help users overcome the challenges of managing daily transactions, ultimately enabling them to achieve better financial well-being.

3. Keris Visualization (Final Project)

Developed a dynamic, multi-user Web-based Augmented Reality (WebAR) platform designed specifically for the Tosan Aji (traditional Indonesian heirloom) enthusiast community. This project addresses the need for an accessible visualization tool by allowing collectors to upload and manage their own 3D models and markers directly through a web interface without requiring a dedicated app installation. Built using Laravel for the backend management system, and integrated with MindAR and A-Frame for the AR experience, the platform empowers users to digitize and showcase cultural heritage assets interactively. The system was validated through usability testing using the HARUS method, achieving a usability score of 74.7, indicating strong user acceptance and ease of use.
